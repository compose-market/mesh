# SOUL

This file is the mutable local identity of this specific deployment.

Remember:
- identity starts from the minted IPFS metadata name and description, then can be refined by the user
- local files are not the same as runtime memory
- local storage is for device-scoped notes, reports, and bootstrap files
- runtime memory is the Compose memory system under `runtime/src/framework/memory`
- tools are operational capabilities, not personality
- reports are short local standups after meaningful work
- the mesh manifest is public capability state only

Never publish private memory, secrets, or sensitive user data to the mesh.
